# Customer Service Chatbot Challenge

## Problem Statement

Suppose, as part of a data science team at a financial institution, you have been tasked with improving customer experience by automating the tagging of client queries using a machine learning model. The queries are short pieces of text which are received via a customer service chatbot. The customer queries need to be tagged so that they can be routed to appropriate departments for further investigation.

## Data

You have been provided with a dataset consisting of labeled queries (file: `ds_task_dataset.csv`). There are ten categories for the queries. The categories and their associated encoding are listed in the table below.

| Label | Encoding |

| DIRECT_DEBIT_PAYMENT_NOT_RECOGNISED | 1 |
| BALANCE_NOT_UPDATED_AFTER_CHEQUE_OR_CASH_DEPOSIT | 2 |
| WRONG_AMOUNT_OF_CASH_RECEIVED | 3 |
| CASH_WITHDRAWAL_CHARGE | 4 |
| TRANSACTION_CHARGED_TWICE | 5 |
| DECLINED_CASH_WITHDRAWAL | 6 |
| TRANSFER_FEE_CHARGED | 7 |
| TRANSFER_NOT_RECEIVED_BY_RECIPIENT | 8 |
| BALANCE_NOT_UPDATED_AFTER_BANK_TRANSFER | 9 |
